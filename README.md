📊 Startup Success Prediction using Machine Learning
📌 Project Overview

This project predicts whether a startup will be successful or not using Machine Learning techniques.
It involves data preprocessing, feature selection, model building, optimization, and deployment.

The system analyzes startup-related data and generates predictions that help in decision-making.

🚀 Features

Data preprocessing and cleaning

Exploratory Data Analysis (EDA)

Feature selection

Model training using Random Forest

Model evaluation and optimization

Prediction using trained model

Simple executable application

📂 Dataset

The dataset contains startup details such as:

Funding

Market size

Team size

Revenue

Growth metrics

These features are used to predict startup success.

🧠 Machine Learning Model

Algorithm used:
